Module: clone()

Créé une copie du tableau non-profonde du tableau.
Source:
  • Main.js, line 3

Examples

let a = [1,2,3], b=a.clone();
a[0] = 0;
console.log(b); // [1,2,3] (copie des valeurs)
let obj1 = {a:1}, obj2 = {b:2};
let a = [obj1, obj2];
let b = a.clone();
console.log(b); //[ { a: 1 }, { b: 2 } ]
obj2.b = 0;
console.log(b); //[ { a: 1 }, { b: 0 } ] (copie des références)
